  • Visualizing simulations

    Use the basic plotting tool to preview simulations. Use the Python API to construct more advanced visualizations (see examples). Note: The Python API is not publicly accessible. Researchers must install WholeCellSimDB on their own machines to use the Python API.

  • Exporting data

    Click the icons at the bottom-right to export the data described on each page. Use the "Download" menu at the top-left to download the entire database.
