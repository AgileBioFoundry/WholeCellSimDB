About WholeCellSimDB

WholeCellSimDB is a database of whole-cell model simulations designed to make it easy for researchers to explore and analyze whole-cell model predictions including predicted:

  • Metabolite concentrations,

  • DNA, RNA and protein expression,

  • DNA-bound protein positions,

  • DNA modification positions, and

  • Ribome positions.

Getting started?

Please view the tutorial and help pages for help getting started.

Implementation

WholeCellSimDB is a hybrid SQL/HDF database implemented in Python. The Django framework was used to construct a MySQL database containing simulation metadata as well as links to HDF files containing the simulation predictions. The H5py package was used to read and write HDF files. Full text search over the simulation metadata was implemented using Xapian and Haystack. The web interface was run using Apache and mod_wsgi. The web interface was developed using flot, Google Fonts, jQuery, jqTree, jQWidgets, and FamFamFam Silk icons.
